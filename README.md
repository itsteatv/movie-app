# 🎥 Movie App
It's Movie App but slightly different 🔥

## 🖥️ Run Locally

1️⃣ Clone the project

```bash
  git clone https://github.com/itsteatv/movie-app.git
```

2️⃣ Go to the project directory

```bash
  cd movie-app
```

3️⃣ Install dependencies

```bash
  npm install
```

🔚 Start the server

```bash
  npm run dev
```
